# Asiri Intelligence OS v1

Independent research and portfolio-decision workspace inside `ASIRI-2026`.

## Runtime architecture

Verified provider health + verified OHLCV snapshots → Data Quality Gate → Stock and Portfolio Intelligence → transparent read-only Decision Engine.

## v1 modules

1. Market Intelligence.
2. Portfolio Intelligence.
3. Stock Intelligence.
4. Opportunity Intelligence.
5. Decision Engine.
6. Data Quality Engine.

Portfolio data stays in browser localStorage. No broker connection, credentials, or automatic order execution.
