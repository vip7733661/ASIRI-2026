# Asiri Execution Desk — Saxo SIM

مساحة تنفيذ تجريبية معزولة داخل `ASIRI-2026`.

## المسار

- محليًا: `/execution-desk/`
- GitHub Pages: `https://vip7733661.github.io/ASIRI-2026/execution-desk/`

## ما يعمل الآن

- قراءة `market-data.json` المنشور من Asiri Intelligence OS.
- إنشاء أمر Limit تجريبي.
- حساب قيمة الصفقة والخسارة القصوى ونسبة المخاطرة وR:R.
- تطبيق حد أقصى لقيمة الأمر والمخاطرة اليومية وKill Switch.
- منع الأوامر المتطابقة.
- تأكيد يدوي بكتابة كلمة `تأكيد`.
- تخزين أوامر SIM وسجل التدقيق محليًا على الجهاز.
- محاكاة قبول أو إلغاء الأمر دون الاتصال بوسيط.

## ما لا يعمل عمدًا

- لا يوجد اتصال بحساب Saxo.
- لا توجد مفاتيح أو Tokens في الواجهة.
- لا يتم إرسال أوامر إلى SIM أو LIVE.
- لا يتم حل رمز السهم إلى Saxo UIC حتى الآن.
- لا يوجد تنفيذ تلقائي.

## المرحلة التالية

Backend خاص يدعم:

1. OAuth Authorization Code مع Saxo SIM.
2. تخزين Tokens مشفرًا على الخادم.
3. قراءة الحساب والمراكز والأوامر.
4. البحث عن الأداة والتحقق من UIC وقيود التداول.
5. Pre-check للأمر.
6. إرسال أمر SIM بعد التأكيد اليدوي فقط.
7. متابعة الحالة ومنع التكرار باستخدام `x-request-id` وسجل تدقيق دائم.

أي انتقال إلى LIVE يتطلب اختبار SIM مستقلًا، وموافقة صريحة، وضوابط مخاطر إضافية.
