# API Atlas — Finance Edition

تطبيق ويب تقدمي (PWA) مستقل مبني فوق عينة من قسم **Finance** في مستودع `public-apis/public-apis`.

## ما الذي يعمل؟

- البحث بالاسم والوصف والوسوم.
- التصفية حسب نوع الخدمة والمصادقة وCORS والمفضلة.
- تقييم أولي لسهولة التكامل، لا لجودة البيانات.
- حفظ المفضلة محليًا على الجهاز.
- تنزيل الدليل بصيغة JSON.
- مشاركة التطبيق.
- العمل دون اتصال بعد أول زيارة.
- التثبيت على Android، والإضافة إلى الشاشة الرئيسية على iPhone.

## التشغيل محليًا

```bash
python3 -m http.server 4174 --directory api-intelligence
```

ثم افتح:

```text
http://localhost:4174
```

## التثبيت على iPhone

1. افتح التطبيق في Safari.
2. اضغط زر المشاركة.
3. اختر **إضافة إلى الشاشة الرئيسية**.
4. اضغط **إضافة**.

## حدود النسخة MVP 0.1

- التطبيق دليل بحث يعمل ببيانات مدمجة، وليس مراقبًا حيًا لصحة الخدمات.
- حالة CORS والخطط المجانية وشروط الاستخدام قد تتغير.
- لا يتم إرسال مفاتيح API أو تخزينها.
- لا يوجد تداول أو اتصال بوسيط.
- Catalog Score يقيس سهولة التكامل الأولي فقط.
